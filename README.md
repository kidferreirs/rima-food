# 🍔 Rima Food

> Sistema SaaS inteligente para restaurantes desenvolvido pela **Rimatech**.

---

# 🚀 Sobre o projeto

O **Rima Food** é uma plataforma completa de gestão para restaurantes, desenvolvida para centralizar atendimento, pedidos, operação da cozinha, cardápio digital, CRM e automações com Inteligência Artificial.

Mais do que um ERP, o Rima Food conecta todos os setores do restaurante em um único ecossistema.

---

# 🎯 Nossa missão

Facilitar a operação dos restaurantes utilizando tecnologia, automação e Inteligência Artificial, permitindo que os clientes atendam melhor, vendam mais e tenham controle total da operação.

    ▼

    ▼
📦 Pedidos
    │
    ▼
⚙️ PedidoEventService
